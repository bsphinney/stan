## godmode v0.1.0 — new program

Separate Mac app at `~/Documents/godmode/`. GitHub:
`https://github.com/bsphinney/godmode` (private).

Does:
- Reads `/Volumes/proteomics-grp/STAN/<instrument_id>/` (SMB mirror)
- FastAPI dashboard on port 8431 with inline React HTML
- CLI: `godmode list|serve|render|export|version`

To launch after reboot:
```bash
cd ~/Documents/godmode
.venv/bin/godmode serve --root /Volumes/proteomics-grp/STAN
# Open http://127.0.0.1:8431
```

If the venv is wedged again: `.venv/bin/pip install -e . --force-reinstall --no-deps`
and retry. Nuke-and-recreate recipe:
```bash
rm -rf .venv && python -m venv .venv
.venv/bin/pip install -e .
```
